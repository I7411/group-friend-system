# Group Management and Friend Recommendation System

## Overview

This is a System Analysis and Design course project for a social network group management and friend recommendation system.

The project focuses on analyzing and designing a web-based system that supports users in creating groups, joining communities, interacting with group members, and receiving friend recommendations based on user-related information such as mutual friends, common groups, interests, profile information, and interaction level.

## Course Information

- Course: System Analysis and Design
- Project Type: Academic Project
- System Type: Web-based Social Network System
- Role: Business Analyst / System Analyst

## Objectives

The main objectives of this project are:

- Analyze the business requirements of a social network group management system.
- Model core business processes using UML diagrams.
- Design system functions for group management and friend recommendation.
- Define actors, use cases, workflows, and data relationships.
- Design user interfaces and database structure for the proposed system.

## Main Features

### User Management

- Register and log in to the system.
- View and update personal profile.
- Manage privacy settings.
- Protect user accounts.

### Group Management

- Create new groups.
- Update group information.
- Search for groups.
- Send requests to join groups.
- Approve or reject group joining requests.
- Manage group members.
- Assign member roles and permissions.
- Remove or block violating members.

### Group Interaction

- Create posts in groups.
- Comment and interact with group content.
- Review and approve posts.
- Hide or remove inappropriate posts.

### Friend Recommendation

- View suggested friends.
- Send friend requests.
- Accept or reject friend requests.
- Block users.
- Recommend friends based on mutual friends, common groups, interests, profile information, and interaction level.

### Violation Report Management

- Report inappropriate users, posts, or behaviors.
- Process violation reports.
- Notify users about report handling results.

This project focuses on system analysis and design only. It does not include full implementation of a real social networking platform.

The project does not cover advanced features such as:

- Real-time chat
- Voice call or video call
- Livestream
- Online payment
- Marketplace
